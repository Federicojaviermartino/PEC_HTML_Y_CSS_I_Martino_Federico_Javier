Accesibilidad

Este sitio web ha sido desarrollado siguiendo las pautas de accesibilidad WCAG 2.0 nivel AA. Se han implementado las siguientes características:

  • HTML semántico con etiquetas apropiadas
  • Atributos alt descriptivos en todas las imágenes
  • Contraste de color adecuado (mínimo 4.5:1)
  • Enlaces "saltar al contenido principal"
  • Navegación por teclado funcional
  • Atributos ARIA donde son necesarios
  • Etiquetas lang correctamente especificadas

Sobre este Proyecto

Este sitio web ha sido desarrollado como proyecto educativo para la asignatura HTML y CSS I del Máster en Desarrollo de Aplicaciones y Sitios Web de la UOC (Universitat Oberta de Catalunya).

Fecha de realización: Octubre-Noviembre 2024

Propósito: Demostrar competencias en desarrollo frontend moderno, diseño responsive, accesibilidad web y uso de herramientas de construcción.
