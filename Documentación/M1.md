M1. Las herramientas para desarrolladores de los navegadores
Este módulo da inicio a la asignatura Herramientas HTML y CSS. Durante este módulo nos familiarizaremos con las herramientas para desarrolladores que incorporan por defecto los navegadores web. En esta guía de estudio se exponen los objetivos de aprendizaje y se proponen diferentes etapas, recursos y actividades para alcanzar estos objetivos.

Durante los primeros años del desarrollo web, el análisis del código y de posibles errores se tenía que hacer manualmente. Si bien algunos navegadores incorporaban una consola de JavaScript, no fue hasta la aparición de la extensión Firebug para el navegador Firefox, en 2006, cuando los navegadores dispusieron de herramientas que permitían inspeccionar y depurar el código HTML, CSS y JavaScript.

Firebug ha sido sin duda la piedra fundacional de lo que conocemos hoy en día como las herramientas para desarrolladores de los navegadores. Actualmente ya vienen incorporadas de forma nativa en los principales navegadores, como Firefox, Chrome, Edge o Safari, y son la herramienta básica para los desarrolladores web front-end.

En este módulo conocemos cómo funciona el inspector, el editor de estilos y de las herramientas para diseño responsive, tres de las utilidades básicas que conforman las herramientas para desarrolladores más habituales de los navegadores. Otras herramientas que se incorporan habitualmente y que también os animamos a explorar y conocer son: la consola de JavaScript o los inspectores de red y de rendimiento.

Los materiales de aprendizaje los encontraréis en la sección Recursos básicos. En la sección Contenidos y actividades encontraréis diferentes ideas y propuestas de actividades sencillas para consolidar los conocimientos adquiridos con los recursos propuestos. Os animo a que los hagáis y si os surge alguna duda sobre las actividades propuestas, abráis un hilo en el foro y lo comentemos entre todos.

Objetivos de aprendizaje
Conocer y familiarizarse con las herramientas para desarrolladores de los navegadores web.
Conocer el funcionamiento y la utilidad del inspector.
Conocer el funcionamiento y la utilidad del inspector de Flexbox y Grid.
Conocer el funcionamiento y la utilidad del editor de estilos.
Conocer el funcionamiento y la utilidad de las herramientas para diseño responsive.
Recursos básicos
Para la realización de las actividades de este módulo podéis usar cualquier navegador web moderno que disponga de herramientas para desarrolladores. Recomendamos especialmente (sobre todo de cara a futuras actividades) descargar el navegador Firefox Developer Edition para realizarlas. Esta edición especial del navegador Firefox incorpora herramientas específicamente pensadas para desarrolladores, así como la incorporación de los estándares web más recientes y características experimentales.

Actualmente existe una fuerte competencia para atraer desarrolladores web a cada uno de los ecosistemas de los navegadores, lo que implica que estas herramientas evolucionan a un ritmo muy rápido. Aunque las herramientas básicas son las mismas, según el navegador que utilicéis dispondréis de herramientas complementarias o características específicas que solo están presentes en ese navegador. Por ejemplo, en el caso de Firefox, claramente tiene las mejores herramientas para depurar Flexbox, CSS Grid o los cambios de CSS en vivo. Es necesario que conozcáis qué herramientas incorpora cada navegador, los puntos fuertes de las mismas, y también de las novedades que van incorporando. Por eso os sugerimos que estéis atentos a las novedades que se van publicando sobre las herramientas para desarrolladores de FirefoxLinks to an external site. y de ChromeLinks to an external site.. Las actividades de este módulo las realizaremos sobre las herramientas que ofrece Firefox Developer Edition, que os animamos a descargar.
